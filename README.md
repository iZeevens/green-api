## Локальный запуск проекта (Vite)

### Требования
- **Node.js** (рекомендуется версия LTS)
- **npm** (идёт вместе с Node.js) или **yarn/pnpm**

### Установка и запуск

1. Установите [Node.js](https://nodejs.org/) (если ещё не установлен).
2. Установите зависимости:
   ```sh
   npm install
   ```
3. Запустите сервер:
   ```sh
   npm run dev
   ```
